The Airport Simulator simulates an airport runway on the console, supporting things like adding/removing planes that are taking off, with their details and etc. This is a learning project I did to learn how Stacks/Queues are implemented in Java and to understand them as data structures in general. Planes taking off and landing are managed by two queues, with helper functions implemented as needed.

How it works.

Features:
● Adding/Removing planes to/from the take-off queue.
● Adding/Removing planes to/from the landing queue.
● Queueing the next step for a flight, e.g. take-off or landing.
● Printing all the data in the queues.
● Outputting all the data in the queues to a file.
